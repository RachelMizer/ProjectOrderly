3. Managing Orders

Viewing the Order List

placeholder

The Orders section displays all orders placed through your store in a sortable table. Each row represents a single order and shows five pieces of information: the Order #, the Date the order was placed, the Customer name, the order's current Status, and the order Total.

Every order carries one of three statuses. Pending orders have been placed but not yet fulfilled. Completed orders have been marked as fulfilled. Cancelled orders were called off before completion. Each status is displayed as a colour-coded badge so you can read the state of your orders at a glance.

You can sort the list by any column by clicking its header. Click the same header again to reverse the sort direction. An arrow indicator in the column header shows which column is active and whether it is sorted ascending or descending. By default, the list is sorted by date with the most recent orders at the top.

At the bottom of the list, Orderly shows how many orders are currently displayed and provides pagination controls — First, Prev, Next, and Last — to move through large order histories. The current page and total page count are shown between the controls.

For any order with a Pending status, a Mark Complete button appears in the Actions column. Clicking it marks the order as fulfilled without needing to open the order detail — useful for quickly processing a batch of orders.

Filtering Orders

placeholder

The toolbar at the top of the Orders section contains several tools for narrowing down the order list. You can use them individually or in combination.

The search bar filters the list as you type, matching against order number and customer name. It is useful for jumping directly to a specific order or finding all orders from a particular customer.

The Status dropdown lets you view only orders of a particular status — Pending, Completed, or Cancelled. Select All Statuses to return to the full list.

The date dropdowns let you filter by Year, Month, and Day. They work in sequence — you must select a year before a month becomes available, and a month before a day becomes available. You can stop at any level: filtering by year alone shows all orders from that year, filtering by year and month narrows to that month, and so on. The page title updates to reflect the active date range, for example Orders from March 2026 or Pending Orders from March 5, 2026.

When any filter is active, a Clear Filters button appears in the toolbar. Click it to remove all filters and return to the full, unfiltered order list.

Opening an Order Detail

placeholder

To view the full details of an order, click its order number in the list. This opens the order detail page, which gives you a complete breakdown of everything associated with that order.

At the top of the page you will find the order's key information at a glance: the order number, the date and time it was placed, the customer's name, and the current status badge. For Pending orders, the Mark Complete button also appears here.

Below the header, the Order Details table lists every item in the order. Each row shows the product name, the selected variant, any modifiers that were applied (such as add-ons or customisations), the quantity ordered, the unit price charged, and the line total. At the bottom of the table, Orderly displays the order subtotal, the tax amount, and the final total due.

To return to the order list, click Back to Orders in the toolbar. You can also print the order detail page using the Print button in the same toolbar.

Changing Order Status

placeholder

Every order in Orderly has a status that reflects where it is in its lifecycle: Pending, Completed, or Cancelled. You can update an order's status from either the order list or the order detail page.

To mark an order as complete, click the Mark Complete button next to the order. This moves the order from Pending to Completed and signals that the order has been fulfilled. Once marked complete, the button is removed and the status badge updates accordingly.

To cancel an order, open the order detail page and click the Cancel Order button. Cancelling an order moves it to a Cancelled status. Cancelled orders remain visible in the order list and can be filtered by status, but no further actions can be taken on them.

If an order has been marked complete in error, you can revert it to Pending using the Revert to Pending button on the order detail page. This returns the order to an active state so it can be fulfilled or cancelled as needed.
