# Orderly — Page & Permission Map
### Created by Rachel Mizer
This document outlines all pages in the Orderly system and specifies whether each page is accessible to **Admins** or **Users**.
Orderly uses a simple two‑role permission model:

- **Admin** — full system access  
- **User** — standard business user with limited capabilities  

---

## 1. Authentication
| Page | Access |
|------|--------|
| Login | Public |
| Logout | Logged‑in users |
| Password Reset | Public |
| Email Verification | Public |

---

## 2. Dashboards
| Page | Access |
|------|--------|
| **Admin Dashboard** | **Admin‑only** |
| User Dashboard | User |

---

## 3. Catalog (Products)
| Page | Access |
|------|--------|
