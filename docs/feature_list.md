# Feature List Based on the SRS #

## 1\. User Management

*   Secure login and registration
*   Email verification after account creation
*   Password reset and account recovery
*   Guest checkout (ordering without an account)
*   Role-based permissions (Customer vs. Business Admin)
*   Automatic session timeout and logout

## 2\. Product Browsing & Details

*   Product catalog browsing
*   Product category filtering
*   Product detail view (name, description, price, image)
*   Product search (basic keyword search)
*   Product customization options (if applicable)

## 3\. Ordering System

*   Add items to order
*   Customize order items
*   Submit order
*   Order confirmation screen
*   Order status tracking (Pending → In Progress → Ready → Completed)
*   Order status timeline display
*   Customer order history
*   Customer order cancellation (while Pending)

# 4\. Inventory & Product Management (Admin)

*   Manage products (add, edit, delete)
*   Manage pricing
*   Update and manage inventory levels
*   Low-stock notifications
*   Bulk product import via CSV
*   Product category management

## 5\. Administrative Dashboard

*   Dashboard homepage with key business metrics
*   View sales summaries
*   Export sales summaries as spreadsheets
*   View and filter orders (by date, status, customer)
*   Activity log viewer (audit logging UI)
*   Supplier information management (contacts, details)
*   Business configuration settings (tax rates, service fees, etc.)

## 6\. Reporting

*   Sales summary reports
*   Inventory reports
*   Order reports
*   Exportable formats (CSV, XLSX)

## 7\. System Infrastructure & Configuration

*   Core navigation layout (top bar, sidebar, submenu)
*   Responsive web design for cross-device compatibility
*   API communication layer
*   Authentication handling (tokens, protected routes)
*   Error handling and empty-state messaging
*   Loading states for data-heavy screens
*   Accessibility improvements (ARIA labels, keyboard navigation)

## 8\. Help & Support

*   User help / FAQ page
*   Onboarding or guidance for new users (optional but mentioned)
